How to connect Power Automate

  1. Create an inbound webhook here — choose the action Power Automate should trigger
  2. Copy the Endpoint URL shown on the webhook card
  3. In Power Automate, add an HTTP action (POST) pointing to https://your-domain/api/inbound/<token>/
  4. Set the body to JSON matching the expected payload for the chosen action
  5. Run the flow — NPI Tracker will process the event automatically
