Inbound Webhooks

Setup guide — Teams Workflows (free, recommended by Microsoft)

⚠️ The old "Connectors → Incoming Webhook" method is being retired by Microsoft. Use Workflows instead.

  1. In Teams, go to the channel where you want notifications
  2. Click ··· next to the channel name → select Workflows
  3. Search for "Post to a channel when a webhook request is received" and select it
  4. Give the workflow a name → click Next
  5. Select the Team and Channel to post to → click Add workflow
  6. Copy the generated webhook URL — it starts with https://prod-xx.logic.azure.com/…
  7. Click New Webhook here, paste the URL, choose an event → Create
  8. Click Test — an Adaptive Card should appear in your Teams channel
